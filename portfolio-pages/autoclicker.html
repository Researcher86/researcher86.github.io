Автокликер

Разработал программу "Автокликер" для приемной комиссии Костанайского инженерно-экономического университета им. М. Дулатова. Программа была разработана для автоматизации процесса переноса информации по абитуриентам университета, из внутренней системы в стороннюю систему. Так как сторонняя система не поддерживает Web API, пришлось реализовать взаимодействие с системой по средствам UI интерфейса (через веб-форму). Используемые технологии: Java, Selenide, Spring, SLF4J (Logback), Maven, MySQL.

Веб-форма сторонней системы для добавления информации.
