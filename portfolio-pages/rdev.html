Веб-приложение для системных аналитиков

Принимал участие в разработке веб-приложения для системных аналитиков. Система позволяет системным аналитикам создавать стандартное CRUD приложение, по средствам написания JSON схемы структур хранения данных. В JSON схеме было реализовано наследование структур данных и создание собственных типов данных. На основе данной схемы, система генерировала структуру БД и предоставляла веб-интерфейс, для предоставления пользователю возможности ввода данных в систему. Для реализации работы CRUD с динамически сгенерированной структурой хранения данных, использовался протокол OData v4.> Используемые технологии: JS (dxDataGrid), Java, Spring Boot, Thymeleaf, JUnit, Maven, PostgreSQL. Приложение разработано в ООО "Интегрикс" для системных аналитиков компании.

  3. фильтр

  2. выбор значения из справочной структуры/таблицы данных.
