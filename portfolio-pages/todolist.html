ToDoList

Разработал программу "ToDoList" для ИТЦ отдела Костанайского инженерно-экономического университета им. М. Дулатова. Программа была разработана для автоматизации процесса контроля назначения, выполнения заданий для сотрудников отдела. Используемые технологии: Java 8, Thymeleaf, Spring, JUnit, Mockito, Maven, JPA/Hibernate, WebSocket, Twitter Bootstrap, JQuery, Tomcat, MySQL, HSQLDB.
