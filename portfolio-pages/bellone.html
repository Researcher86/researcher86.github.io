BellOne

Принимал участие в разработке микро сервисной системы USS в компании ООО "Бэлл Интегратор". Система разворачивалась на серверах AWS. Моя задача была работать над сервисами upload-service – сервис предоставлял пользователю производить загрузку файлов в хранилище AWS S3, для дальнейшей тренировки нейронной сети, и сервисом csi-service – этот сервис агрегирует в себе всю логику по работе с пользовательскими данными, подпиской, покупкой, предоставляемых сервисов/услуг системы USS. Технологический стек: Java 8, JUnit, Spring Boot(core, mvc, data, cloud), JPA/Hibernate, Flyway, Maven, PostgreSQL.
