Веб-портал

Разработал информационную систему "Веб-портал" http://portal.kineu.kz на языке программирования PHP, JavaScript. Движок сайта был реализован по шаблону MVC. Для предотвращения частой перезагрузки страницы применил асинхронные запросы AJAX. Использовал: Twitter Bootstrap, JQuery, CKEditor, ЧПУ, PDO, MySQL (хранимые процедуры, представления, подзапросы, объединения, индексы). Применил нормальные формы проектирования БД. Программа была разработана для Костанайского инженерно-экономического университета им. М. Дулатова.

Главная страница портала. Здесь отображается список всех объявлений. Любой сотрудник может здесь оставить свое объявление.

Страница с заданиями. Здесь выводятся все задания конкретного пользователя. Пользователь может просматривать входящие, исходящие задания, а также может следить за статусом выполнения порученных заданий.

Страница отправки нового здания. Здесь сотрудник может дать новое здание любому сотруднику организации.

Список сотрудников. Из данного списка сотрудник может выбрать либо конкретного сотрудника, либо подразделение, для кого предназначено здание.

Папка задания. Переход в данный раздел осуществляется автоматически, если пользователь установил галочку "Прикрепить файлы". В данном разделе могут добавляться файлы с заданием, либо файлы с выполненными заданиями, только тех пользователей, которые были указаны в качестве адресатов.

Страница расписания. Данный раздел формируется динамически и отображает список файлов который находятся на внутреннем сервере. Было необходимо предоставить доступ к этим файлам из интернета.
