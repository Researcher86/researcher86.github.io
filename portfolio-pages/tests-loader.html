TestsLoader

Разработал программу на Delphi для полуавтоматической загрузки тестовых заданий в базу данных. Программа считывает документ в формате *.doc, *.docx, создает временную БД MS Access, генерирует HTML для пред-просмотра и если пользователя все устраивает, грузит все тесты в БД MySQL. Программа была разработана для Костанайского инженерно-экономического университета им. М. Дулатова.
