Антиплагиат

Разработал систему "Антиплагиат". База данных была реализована на MySQL, основная логика информационной системы была реализована на серверном языке программирования PHP, для удобства и оптимизации работы пользователя, была реализована клиентская часть приложения на языке программирования Java (Swing). Были применены полнотекстовый поиск и алгоритм шинглов. Для взаимодействия клиентской и серверной частей системы был использован архитектурный стиль REST. Система была разработана для Костанайского инженерно-экономического университета им. М. Дулатова.

Веб интерфейс серверной части системы. Был реализован для предварительной проверки документов всеми желающими.

Интерфейс клиентского приложения ИС "Антиплагиат". Данный модуль системы позволяет:

  • Провести проверку текста
  • Организовать пакетную проверку файлов, и не принуждает ожидать окончания полной проверки, после проверки результаты автоматически сохраняются в базу данных
  • Пополнять базу данных. Каждый документ перед загрузкой в БД проходит проверку
  • Просмотреть результаты проверки документов
  • Экспорт результатов в Excel
